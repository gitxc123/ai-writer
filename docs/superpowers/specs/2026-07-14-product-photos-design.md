# 产品介绍 · 实物图增强管线

**日期：** 2026-07-14  
**状态：** 待用户确认  
**范围：** 仅「产品介绍」模板；其它模板的 AI/网络配图逻辑不变

## 背景

当前「产品介绍」只有文案字段 + 通用配图（文生图或搜图），无法基于用户上传的实物照片生成保真图。本功能补齐：上传三视图 → 画质修复 → 多角度特写 → 应用场景匹配，并同步生成产品介绍文案。

## 已确认决策

| 项 | 选择 |
|----|------|
| 上传 UI | 三格横排：正面 / 侧面 / 细节 |
| 最少上传 | ≥ 1 张即可生成 |
| 交付量 | 标准档，约 6–8 张 |
| 技术方案 | Agnes **图生图（img2img）** 管线 |
| 触发范围 | 仅模板名 = `产品介绍` |

## 用户流程

1. 打开「产品介绍」创作页  
2. 填写产品名称/卖点、目标人群、字数（沿用现有字段）  
3. 在三格槽位上传 1–3 张实物图  
4. 点击生成 → 异步任务（仍须有效会员）  
5. 任务页查看：文案 + 标注类型的图片集，可一键复制图文  

## 交付构成（标准档）

| 类型 | 数量 | 说明 |
|------|------|------|
| `enhanced` | 与上传张数相同（1–3） | 图生图修画质，保留主体 |
| `closeup` | 2–3 | 多角度/特写，以上传图为参考 |
| `scene` | 2 | 按产品 + 目标人群匹配应用场景 |

合计约 **5–8** 张（上传 1 张时偏少端，上传 3 张时偏多端）。未上传的槽位不生成对应 `enhanced` 图。

## UI 设计

### 创作页（`create.vue`）

- 当 `template.name === '产品介绍'` 时：
  - 展示三格上传区（标签：正面 / 侧面 / 细节）
  - 隐藏通用「配图数量 / AI·网络搜图 / 尺寸」控件（改由管线决定产出）
- 其它模板：行为与现网一致  
- 校验：未上传任意图时禁止提交，提示「请至少上传 1 张产品图」  
- 单张限制：建议 ≤ 5MB；格式 jpg/png/webp  

### 任务结果页

- 图片按 meta 类型分组或角标显示：已修复 / 特写 / 场景  
- 复制打包时带上全部生成图（沿用现有平台导出能力）

## 后端架构

```
上传 ──► uploads/ + 公网 URL
              │
生成任务 ──► input.productPhotos [{ slot, url }]
              │
         ┌────┴────┐
         │ 文案生成 │（并行）
         └────┬────┘
              │
         图片管线（串行或小并发）：
         enhance(each upload) → closeup×2–3 → scene×2
              │
         imageUrls[] + imageMeta[{ type, slot?, caption, url }]
```

### 新增 / 变更接口

| 接口 | 作用 |
|------|------|
| `POST /api/uploads/product` | multipart 上传；鉴权；返回 `{ slot?, url }` |
| `POST /api/generate` | 产品介绍时要求 `productPhotos`；忽略/覆盖原 `imageSource` 走专用管线 |

### Agnes

- 复用现有 Key；扩展 `agnes-image.js`：支持 img2img（参考图 URL + 英文编辑指令）  
- 修画质 prompt：增强清晰度/光照，保留产品形状与品牌特征，无文字叠字  
- 特写 / 场景 prompt：保留产品外观，改变角度或放入匹配场景  

### 存储

- 本地目录 `backend/uploads/`（或 `public/uploads`），Express 静态托管，保证 Agnes 能拉到 HTTPS/公网 URL；本地试用可用临时图床作为降级  
- 上传 URL 写入任务 `input` JSON，不新增 Prisma 列（YAGNI）

### 错误处理

- 单张 img2img 失败：记录该条失败，尽量继续其余张；全部失败则任务 `failed` 并带部分文案（若文案已成）或整体失败  
- 上传非法格式/超限：接口 400  

## 非目标（本版不做）

- 真实微信/支付宝支付对接  
- 非「产品介绍」模板的实物上传  
- 视频 / 3D  
- 手动微调每一张 prompt 的进阶编辑器  

## 验收标准

1. 产品介绍页可见三格上传，其它模板无此 UI  
2. 只传 1 张也能完成任务，产出修复图 + 特写 + 场景（合计约 5–6 张量级）  
3. 生成图明显以用户实物为参考（非无关插画）  
4. 任务结果含文案与分类后的图片  
5. 小红书等模板配图逻辑未被破坏  

## 测试要点

- 1 / 2 / 3 张上传各跑通一次  
- 0 张被前端/后端拒绝  
- 非会员仍被 403  
- 非产品介绍模板仍可用原 AI/web 配图
